# 白板手绘风格 SVG 规范

## 画布

- viewBox="0 0 1600 900"（16:9），背景 #FEFEFE 或纯白
- 可加极淡的白板质感：几条 opacity 0.03 的灰色斜擦痕（可选）

## 三色笔

| 用途 | 颜色 | 场景 |
|---|---|---|
| 黑色主体 | #2b2b2b | 主干文字、框、主箭头 |
| 红色重点 | #d63b2f | 关键词圈注、重点下划线、感叹标注、"！" |
| 蓝色补充 | #2f5fd0 | 注释、例子、补充说明、次级箭头 |

## 手绘感技法（SVG 实现）

- 文字：中文用 `font-family="'PingFang SC','Hannotate SC','Comic Sans MS',cursive"`，优先 Hannotate SC（手札体，macOS 自带）
- 微旋转：每个文本/组加 `transform="rotate(-1.5 x y)"`，角度在 -3~3 间随机，避免全部水平
- 手绘框：不用规则 rect，用 path 画略微歪斜的四边（顶点偏移 2-5px），stroke-width 2.5-3.5，`stroke-linecap="round"` `fill="none"`
- 波浪下划线：`path d="M x y q 8 -4 16 0 t 16 0 t 16 0"`
- 圈注：椭圆但起终点不闭合（path 画 340° 弧），像随手画的圈
- 箭头：略弯的 quadratic 曲线 + 手绘小箭头（两短线，不用规则 marker）
- 涂改痕迹：一个词上画 2-3 条乱线划掉，旁边用红/蓝写修正词（每图最多 1-2 处，点缀即可）

## 版面结构

- 左上：主题（最大字号 40-48，黑色，可红色波浪线强调）
- 主体：主干 + 分支（类思维导图但允许多中心），或自上而下流程
- 讲解动线：①②③ 圆圈编号（手绘圈）标出阅读顺序
- 右下角或空白处：蓝色小字的"老师碎碎念"式注释 1-2 条
- 留白 ≥ 25%：真白板不会写满
