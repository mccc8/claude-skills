## 步骤
1. 塔尖：把结论压成一句话，**先说**（对方只记得住这句）。
2. 第二层：支撑结论的 2-4 组理由，组内同类、组间互斥。
3. 塔基：每组理由下挂证据（数据/案例/引用）。
4. 反向检查：删掉任何一层，塔尖还成立吗？不成立的才留。
## 强制输出物
`一句话结论 + N 组理由 + 各组证据` 的三层清单，开头永远是结论。
## 接力
结论本身没底 → 先回 analysis-thinking；要落成文档/PPT → 对应产出型 skill
