> 工具卡：冰山模型（轻量单轴版）｜ 何时用：单个事件想快速看深一眼。组织性顽固问题请走 deep-root。 (untools.co/iceberg-model)
## 步骤（四层四问，每层一问）
1. 事件：刚刚发生了什么？（只要事实）
2. 模式：类似的事最近还在哪出现过？频率在变吗？
3. 结构：什么规则/流程/激励让这个模式持续存在？
4. 心智模型：什么被默认相信着（"快比对重要"），才让这个结构合理？
## 强制输出物
四层表 + 一句"当前干预打算落在第几层，为什么够/不够深"。
## 接力
第 3-4 层要动真格（反复≥3次的老毛病）→ deep-root；只需修第 3 层规则 → goal-builder
