# 分析思维 Prompt Pack

本包整理本轮对话中形成的提示词资产，包含：

> **⬆ 本包已升级为可触发的 skill：[analysis-thinking](../../../analysis-thinking/SKILL.md)**——主 SKILL.md 做分诊路由（轻量索引），四份 prompt 成为按需加载的子指令，每次只加载一份。本目录保留原始 prompt 版，供不支持 skill 的普通 AI 工具复制使用。

1. `00_original_mckinsey_style_prompt.md`
   - 原始麦肯锡式分析 prompt 的重构版
   - 用于对比，不作为最终推荐主线

2. `01_analysis_meta_mother_prompt.md`
   - 分析提示词母句
   - 适合放在其他 prompt 前作为总纲

3. `02_simple_general_analysis_prompt.md`
   - 简单版通用分析 prompt
   - 重点：对象分型、切入点、关键断点、最小验证动作

4. `03_standard_general_analysis_prompt.md`
   - 标准版通用复杂对象分析 prompt
   - 重点：对象分型、切入点、断点分型、依赖耦合、解耦重组、行动承接

5. `04_system_analysis_legacy_prompt.md`
   - 旧版系统分析 prompt
   - 重点：切入点、关键断点、关键依赖、耦合点、解耦
   - 适合系统型、业务型、组织型、产品型问题

6. `05_usage_notes_and_comparison.md`
   - 使用建议
   - 与麦肯锡式 prompt 的优劣对比

文件名使用英文，避免部分客户端下载或解压时对中文路径抽风。人类软件生态的礼貌也就到这里了。
